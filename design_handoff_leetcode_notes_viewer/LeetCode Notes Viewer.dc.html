#701 Medium Tree BST leetcode ↗
Insert into a Binary Search Tree
二分探索木（BST）の root と挿入する値が与えられる。挿入後のBSTの root を返す。挿入後もBSTであればどの形でも正解。
Input: root = [4,2,7,1,3], val = 5
Output: [4,2,7,1,3,5]
制約
・ノード数は [0, 10^4]
・-10^8 ≤ Node.val ≤ 10^8、値はすべてユニーク
アプローチ
1 反復による挿入 ▾
思考
  • node と val を比較
  • val が大きければ right、小さければ left へ
  • None に到達したら append して root を返す
実装
class Solution:
    def insert_into_bst(self, root, val):
        if not root:
            return TreeNode(val)
        node = root
        while node:
            if node.val > val:
                if not node.left:
                    node.left = TreeNode(val)
                    break
                node = node.left
            else:
                if not node.right:
                    node.right = TreeNode(val)
                    break
                node = node.right
        return root
Time O(h)
Space O(1)
2 条件式を明示的にして整理 O(h) · O(1) ▸
3 再帰版で実装 O(h) · O(1) ▸
